# Data Reliability Index

The Data Reliability Index package helps engineers treat data trustworthiness as a first-class concern.

## What it provides

- Tiered reliability classification.
- Policy-based filtering.
- Pydantic models for serialization.
- Pandas helpers for analysis pipelines.
- FastAPI examples for ingestion validation.

## Installation

```bash
pip install -e .
pip install mkdocs-material
mkdocs serve
```
