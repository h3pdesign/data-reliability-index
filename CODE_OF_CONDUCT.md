# Code of Conduct

## Our Pledge

We are committed to making participation in this project respectful and constructive for everyone.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Give and receive feedback constructively.
- Keep discussions focused on improving the project.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks or insulting comments.
- Publishing private information without explicit permission.
- Sustained disruption of project discussions.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation when behavior is inconsistent with this code of conduct.

Report conduct concerns privately to the repository owner.
